# Image Processing

## Overview
This repository contains basic image processing code.

## Bilinear interpolation
### Usage
```bash
python src/resize_img.py [-pi <path/to/image>] [-s scale_factor]
```
Note: 
Default scale factor is 2.
Currently only grayscale image supported.
